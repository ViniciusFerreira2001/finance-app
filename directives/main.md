# Finance App Directive

## Goal
Manage and analyze personal or business finances effectively using the 3-layer architecture.

## Context
This instance is designed to automate financial tasks such as:
- Expense tracking and categorization.
- Income reporting.
- Budgeting and forecasting.
- Data visualization in Google Sheets.

## Inputs
- Transaction data (CSV, PDF, or API).
- Budget limits.

## Tools (Execution Scripts)
- None yet. (Scripts will be added to `execution/`)

## Outputs
- Google Sheets dashboards.
- Monthly PDF summaries.

## Edge Cases
- Missing transaction data.
- Duplicate entries.
- Currency conversion issues.
